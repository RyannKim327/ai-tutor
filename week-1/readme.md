# Understanding Basic Logic

In programming, logic is like the way we think about situations in our daily lives. Just like when deciding whether to bring an umbrella, we think: **If it is raining, then I will take my umbrella**.

### Concept Explained

At its core, logic is about making decisions based on facts or statements. For simple logic:
- **True** means something is correct.
- **False** means something is not correct.

We can combine statements to create more complex logic. For example:
- **Statement A**: It is raining.
- **Statement B**: I have an umbrella.

We can combine these statements:
- **Statement A AND Statement B**: It is raining **and** I have an umbrella. This might be true if both conditions are happening.
- **Statement A OR Statement B**: It is raining **or** I have an umbrella. This means at least one of these statements is true.
- **NOT Statement A**: It is **not** raining. This means the opposite of the first statement.

### Real-Life Analogy
Think of a light switch in your home. You can say:
- **The light is ON** (True) or **The light is OFF** (False). 
You can also say:
- **IF the switch is ON, THEN the room is lit up.** This is similar to how programming logic works.
