# Classifying Numbers: Positive or Negative

In this lesson, we will explore how to classify numbers into three simple categories: positive, negative, and zero. This is an important concept because it lays the foundation for many mathematical and programming ideas.

## Understanding Positive and Negative Numbers
To understand this better, think about money:  
- If you have money in your pocket, that’s like a positive number.
- If you owe someone money, that’s like a negative number.
- If you break even (neither owe money nor have any), that’s like zero.

### How to Think About Classifying Numbers
1. **Identify a number**: Start with the number you want to classify. 
2. **Comparison**: Use simple comparisons to decide what category the number falls into.
3. **Logic Steps**: Make decisions on whether the number is positive, negative, or zero using logical steps (if-then scenarios).

### Applying This in "challenge.py"
In your `challenge.py`, you will apply this logical reasoning:  
- **Step 1**: You will first need to think about how to get your number input.  
- **Step 2**: Then, write out the steps you need to follow for classification based on the rules above.  
- **Step 3**: Finally, you will formulate the output to display the classification.
