# Understanding Number Ranges

In this lesson, you're going to learn about ranges of numbers. We often need to check if a number fits within certain boundaries. For example, you might want to see if a person is eligible for a discount if they are making a purchase between $50 and $100. Here, $50 and $100 are limits or boundaries.

## Real-Life Analogy
Imagine you are at a theme park. There might be a sign that says, "You must be between 1.2 meters and 2 meters tall to ride this roller coaster." In this example, the height limits are the boundaries. If you want to ride the roller coaster, you need to check if your height is within this range.

## How to Think
1. **Identify the Number**: Think about the number you want to check. This could be any number.
2. **Set the Limits**: Decide the two numbers that act as fences or barriers. This is your range.
3. **Comparison**: You'll check if your number is above the lower limit and below the upper limit.
4. **Outcome**: Finally, decide what to say based on whether the number falls within these limits.
